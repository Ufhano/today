Today I Learnt

Today I Learnt is a web application that allows users to discover, share, and engage with interesting facts from various topics. The goal is to make learning new things simple, engaging, and community-driven.

Overview

Users can explore a collection of facts contributed by others, add their own, and vote on facts based on how interesting, mind-blowing, or inaccurate they find them. The app promotes curiosity and helps users expand their knowledge every day.

Features

Explore New Facts: Browse a wide range of facts updated regularly.

Add Your Own Facts: Contribute to the community by submitting your own verified facts.

Voting System: Vote facts as Interesting, Mind-blowing, or False.

Category Filtering: View facts by category such as Technology, Science, or History.

Real-Time Updates: Instantly see new facts and vote counts.

Persistent Data Storage: All facts are stored in a database for reliability and longevity.

Tech Stack

Frontend: React 

Backend: Supabase 

Database: PostgreSQL (via Supabase)

Styling: CSS 

How It Works

Users browse existing facts on the home page.

They can submit a new fact along with its source and category.

Each fact displays vote counts for “Interesting,” “Mind-blowing,” and “False.”

The app dynamically updates the facts and votes in real time.

Example

“React is maintained by Meta (formerly Facebook).”
— Category: Technology
— Votes: Interesting: 24 | Mind-blowing: 9 | False: 4
